# Монтаж коротких вертикальных роликов (Shorts / Reels / TikTok)

Полуавтоматический монтаж: распознаём речь → находим повторы/дубли →
ты утверждаешь, что оставить → режем, склеиваем, делаем 9:16, субтитры.

## Что нужно поставить (один раз, Windows / PowerShell)

```powershell
winget install Gyan.FFmpeg
winget install Python.Python.3.12
pip install faster-whisper
```

После установки **закрой и снова открой PowerShell**. Проверка:

```powershell
ffmpeg -version
python --version
```

## Как пользоваться

Положи видео в эту папку. Дальше 3 шага.

### Шаг 1. Распознать речь и найти повторы

```powershell
python transcribe.py myvideo.mp4 --model small --lang ru
```

Создаст рядом:
- `myvideo.transcript.txt` — читаемый текст с таймкодами;
- `myvideo.keep.txt` — **список «что оставить»** (его и редактируем);
- `myvideo.srt` — субтитры;
- `myvideo.transcript.json` — данные для рендера.

Строки с пометкой `[?ПОВТОР]` — это места, где фраза, похоже, проговорена
дважды (неудачный дубль + удачный). Их обычно надо удалить.

### Шаг 2. Отредактировать keep.txt

Открой `myvideo.keep.txt` в блокноте. **Удали строки, которые не нужны**
(в т.ч. помеченные `[?ПОВТОР]`). Каждая оставшаяся строка = кусок, который
войдёт в ролик. При желании подвинь таймкоды вручную (формат `ЧЧ:ММ:СС.ммм`).

### Шаг 3. Собрать ролик

```powershell
python render.py myvideo.mp4
```

Получишь `myvideo.final.mp4` — вертикальный 9:16, лишнее вырезано,
стыки сглажены короткими fade.

Полезные флаги:
- `--no-vertical` — не менять формат кадра (оставить как в исходнике);
- `--fade 0.1` — длиннее плавные переходы на стыках;
- `--burn-subs` — прожечь субтитры в кадр (см. оговорку ниже).

## Субтитры (важно про таймкоды)

`myvideo.srt` сделан по **исходным** таймкодам. Если ты вырезал куски,
субтитры съедут. Чтобы получить **точные** субтитры на смонтированном ролике:

```powershell
# 1) сначала собери ролик без субтитров
python render.py myvideo.mp4
# 2) распознай уже готовый ролик
python transcribe.py myvideo.final.mp4 --model small --lang ru
# 3) прожги субтитры в него
python render.py myvideo.final.mp4 --keep myvideo.final.keep.txt --burn-subs
```

(на шаге 2 keep.txt для final можно не редактировать — просто оставь все строки).

## Подсказки

- Модель `small` — баланс скорость/точность. Для сложного звука — `medium`.
- Если на стыках слышен щелчок/рывок — увеличь `--fade`.
- Музыку, логотип, концевую заставку добавим отдельным шагом — скажи, нужно ли.
